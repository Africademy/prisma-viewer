# Prisma Viewer

## Usage

```
    Prisma Viewer   ಠ_ಠ

    Usage
      $ pv [path]
      
    Options
      -h, --help       Displays this help message
    Examples
      $ pv 
      $ pv ./schema.prisma
```
When viewing the schema/datamodel press the "a" key to change the layout

## Development

`npm run dev [path/to/datamodel]`
